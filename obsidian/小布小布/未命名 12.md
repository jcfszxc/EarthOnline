


```shell

先配置好NapCat。

git clone https://github.com/menghuanan/dynamic-bot.git
cd dynamic-bot/

docker pull menghuanan/dynamic-bot:latest
mkdir -p config data temp logs

替换docker-compose.yml的内容为：

version: '3.8'

services:
  dynamic-bot:
    image: menghuanan/dynamic-bot:latest
    container_name: dynamic-bot
    restart: unless-stopped
    environment:
      - TZ=Asia/Shanghai
    volumes:
      - ./config:/app/config
      - ./data:/app/data
      - ./temp:/app/temp
      - ./logs:/app/logs
    network_mode: "bridge"  # 使用 bridge 网络模式
    logging:
      driver: "json-file"
      options:
        max-size: "10m"
        max-file: "3"
        
        

docker-compose up -d


首次写配置：

# 1. 解除不可变
sudo chattr -i config/bot.yml

# 2. 停止容器
docker-compose stop

# 3. 删掉旧配置，让程序重新生成
rm config/bot.yml

# 4. 启动容器（程序会生成默认 bot.yml）
docker-compose up -d

# 5. 等2秒让程序写完，立刻停止
sleep 2 && docker-compose stop

# 6. 现在编辑配置
nano config/bot.yml
# 修改 host 为 172.17.0.1，first_run_flag 改为 1

# 7. 再次启动
docker-compose up -d
docker-compose logs -f
  
  
之后更新配置：

nano config/BiliConfig.yml
docker-compose restart
docker-compose logs -f




# 停止容器

# 编辑配置确认admin正确
```

分两步走：

**第一步：在目标QQ群里添加订阅**

在群聊里发：

```
/bili add <B站UID>
```

比如要订阅 UID 为 `123456` 的UP主，就在群里发：

```
/bili add 123456
```

这样该UP主的动态和直播都会推送到这个群。

**第二步：确认订阅成功**

```
/bili list
```

查看当前群的订阅列表。

---

**如何找B站UID？**

打开对方B站主页，URL 类似：

```
https://space.bilibili.com/123456
```

`123456` 就是 UID。

---

**如果想推送到私聊而不是群聊**，在私聊窗口里发同样的命令即可，bot 会把动态推送到你的私聊。




**订阅管理**

```
/bili add <UID>        # 添加订阅（在目标群/私聊里发）
/bili rm <UID>         # 删除订阅
/bili list             # 查看当前群/私聊的订阅列表
```

**登录相关**

```
/login                 # B站扫码登录（私聊发给bot）
```

**手动触发**

```
/check                 # 手动检查一次动态（群聊）
```

**群普通管理员**

```
/bili admin add <QQ号>     # 添加本群普通管理员
/bili admin rm <QQ号>      # 移除本群普通管理员
/bili admin list           # 查看本群普通管理员
```

**链接解析黑名单**

```
/bili bl add <QQ号>    # 禁止某人触发链接解析
/bili bl rm <QQ号>     # 解除禁止
/bili bl list          # 查看黑名单
```

**查看帮助**

```
/bili help             # 显示完整帮助图
```

---

**几个注意事项：**

- 所有命令**在哪个群发就作用于哪个群**，私聊发就推送到私聊
- `/bili add` 和 `/bili rm` 必须是**超级管理员**才能用（也就是你）
- UID 是B站主页链接里的数字，如 `space.bilibili.com/123456` → UID是 `123456`
