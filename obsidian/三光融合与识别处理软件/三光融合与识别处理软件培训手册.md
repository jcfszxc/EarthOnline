## 1.2 系统概述

**系统用途**：三光融合与识别处理软件用于海上目标检测识别，通过融合可见光相机、红外相机和激光雷达三种传感器数据，实现对海上船舶等目标的全天候、高精度检测识别。

**源代码特性**：

- **代码规模**：源代码总计约1.5万行，包含12个主要源文件，31个头文件
- **编程语言**：C++14标准，符合ISO/IEC 14882:2014规范
- **架构设计**：采用模块化设计，支持多线程并行处理
- **第三方依赖**：集成OpenCV、GStreamer、RKNN等主流库
- **平台适配**：支持ARM64架构，针对RK3588平台优化

**开发环境与工具链**：

- **开发平台**：Ubuntu 18.04 LTS，支持交叉编译
- **编译器**：GCC 7.5.0，支持C++14特性
- **构建系统**：CMake 3.16，支持多平台构建
- **版本控制**：Git版本管理，支持分支开发

### 5.1.3 详细安装步骤

**步骤1：系统镜像准备**

```bash
# 下载系统镜像
wget https://releases.ubuntu.com/20.04/ubuntu-20.04-server-arm64.iso

# 使用balenaEtcher写入SD卡
# 1. 插入至少16GB的高速microSD卡
# 2. 启动balenaEtcher软件
# 3. 选择下载的镜像文件
# 4. 选择目标SD卡
# 5. 点击Flash开始写入
```

**步骤2：系统初始化**

```bash
# 首次启动后进行系统更新
sudo apt update && sudo apt upgrade -y

# 安装基础开发工具
sudo apt install build-essential cmake git wget -y
sudo apt install python3-pip python3-dev -y
```

**步骤3：OpenCV安装**

```bash
# 安装依赖库
sudo apt install libgtk2.0-dev pkg-config libavcodec-dev libavformat-dev libswscale-dev -y
sudo apt install python3-dev python3-numpy libtbb2 libtbb-dev libjpeg-dev libpng-dev libtiff-dev -y

# 下载OpenCV源码
git clone https://github.com/opencv/opencv.git
cd opencv && mkdir build && cd build

# 配置编译参数
cmake -DCMAKE_BUILD_TYPE=Release \
      -DCMAKE_INSTALL_PREFIX=/usr/local \
      -DWITH_GSTREAMER=ON \
      -DWITH_V4L=ON \
      -DWITH_QT=OFF \
      -DWITH_OPENGL=ON \
      ..

# 编译安装（约需30-60分钟）
make -j$(nproc)
sudo make install
sudo ldconfig
```

**步骤4：RKNN Toolkit 2安装**

```bash
# 下载RKNN Toolkit 2
wget https://github.com/rockchip-linux/rknn-toolkit2/releases/download/v1.5.0/rknn-toolkit2-1.5.0.tar.gz

# 解压安装包
tar -xvf rknn-toolkit2-1.5.0.tar.gz
cd rknn-toolkit2-1.5.0/packages

# 安装Python包
pip3 install rknn_toolkit_lite2-1.5.0-cp38-cp38-linux_aarch64.whl

# 验证安装
python3 -c "from rknn.api import RKNN; print('RKNN installed successfully')"
```

### 5.2.2 工作模式切换操作

**工作模式详解与操作**

|模式名称|功能描述|适用场景|切换指令|参数调整|

### 5.3.3 数据协议理解与应用

**协议帧格式详解**

```
标准数据帧格式：
[帧头2字节][版本1字节][长度2字节][类型1字节][数据N字节][校验2字节][帧尾2字节]

示例：
0xAA55 01 001C 03 [传感器数据] CRC16 0x55AA
```

**协议编程实现**

```c++
// 数据帧结构定义
struct DataFrame {
    uint16_t header;      // 0xAA55
    uint8_t version;      // 协议版本
    uint16_t length;      // 数据长度
    uint8_t type;         // 数据类型
    uint8_t data[MAX_DATA_SIZE];
    uint16_t checksum;    // CRC16校验
    uint16_t footer;      // 0x55AA
};

// 发送数据帧
bool sendDataFrame(int socket, const DataFrame& frame) {
    // 计算校验和
    uint16_t crc = calculateCRC16(frame.data, frame.length);
    
    // 发送数据
    return send(socket, &frame, sizeof(frame), 0) > 0;
}
```
